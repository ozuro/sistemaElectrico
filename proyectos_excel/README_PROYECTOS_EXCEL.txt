Carpeta reservada para respaldos de proyectos y reportes Excel.

Nota:
El sistema HTML/JavaScript guarda el avance en el navegador con localStorage.
Por seguridad, el navegador no puede escribir automaticamente dentro de esta carpeta.

Flujo recomendado:
1. Trabajar el proyecto LP/RP y BT en el mapa.
2. Exportar JSON/KMZ/Excel desde el sistema.
3. Guardar manualmente los archivos descargados en esta carpeta si desea tener respaldo local.
